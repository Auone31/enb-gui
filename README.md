# enb-gui
Personal Project - Designing a GUI

This is still work in progress. It will be a while before it is finalized.

The goal is to efficiently read and display eNB logs. The GUI will be based on C++.

Dependencies:

- gtkmm 3
- g++-6 6.2.0
- GNU Make 4.1
